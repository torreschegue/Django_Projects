# 📚 Descripción del curso

El presente curso tiene como objetivo introducir a los estudiantes en el **desarrollo de aplicaciones web utilizando Python y el framework Django**.

A lo largo del curso se abordarán los fundamentos del desarrollo web moderno, comprendiendo cómo funcionan las aplicaciones web desde el **lado del servidor**, así como la arquitectura y componentes principales que ofrece Django para construir sistemas robustos, escalables y seguros.

Los estudiantes aprenderán a diseñar y desarrollar aplicaciones web mediante prácticas guiadas, proyectos y ejercicios que permitirán comprender el funcionamiento de:

- Proyectos y aplicaciones en Django
- Arquitectura MVC / MTV
- Manejo de rutas (URL Routing)
- Vistas y plantillas
- Modelos y bases de datos
- Formularios y procesamiento de datos
- Buenas prácticas de desarrollo web

---

# 🎯 Objetivos del curso

Al finalizar el curso el estudiante será capaz de:

- Comprender los fundamentos del desarrollo web con Python.
- Crear proyectos y aplicaciones utilizando Django.
- Diseñar interfaces web utilizando plantillas HTML.
- Implementar lógica de servidor mediante vistas.
- Conectar aplicaciones con bases de datos.
- Desarrollar aplicaciones web funcionales siguiendo buenas prácticas.
